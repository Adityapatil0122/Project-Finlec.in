# Finlec.in — Advanced Finance Platform

## 🚀 Live Demo
>  https://finlec.in

## ✨ Features
- 📊 Financial Calculators (SIP, Daily SIP, and more)
- 🔐 Authentication with NextAuth.js
- ⚡ Blazing fast with Next.js Turbopack
- 📱 Fully responsive design
- 🌙 Clean modern UI with Tailwind CSS

---

## 🛠️ Tech Stack
| Technology | Usage |
|---|---|
| Next.js 16 | Framework |
| TypeScript | Language |
| Tailwind CSS | Styling |
| NextAuth.js | Authentication |
| Turbopack | Bundler |

---

## 🏃 Getting Started
```bash
git clone https://github.com/Adityapatil0122/Project-Finlec.in.git
cd Project-Finlec.in
cp .env.example .env.local   # fill in your secrets
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

---

## 🔑 Environment Variables
Create a `.env.local` file:
```
NEXTAUTH_URL=http://localhost:3000
NEXTAUTH_SECRET=your-secret-here
```
